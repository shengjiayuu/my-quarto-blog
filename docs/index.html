Welcome to Python Data Science Tutorial

In this tutorial, we will walk through the steps of using Python to train a machine learning model to predict outcomes based on data.

Step 1: Install Necessary Libraries

To get started, install the required Python libraries by running the following command:

pip install scikit-learn matplotlib pandas

Step 2: Import Libraries

Once the libraries are installed, import them into your Python script:


import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score
import matplotlib.pyplot as plt

Step 3: Prepare the Data

Load your dataset and prepare the input features and target variable.


# Load dataset
data = pd.read_csv('your_data.csv')
X = data.drop('target', axis=1)
y = data['target']

# Split the dataset into training and testing sets
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

Step 4: Train the Model

Now, initialize and train a Random Forest model using the training data.


# Initialize and train the Random Forest model
model = RandomForestClassifier()
model.fit(X_train, y_train)

Step 5: Evaluate the Model

After training the model, evaluate its accuracy using the test dataset.


# Make predictions
y_pred = model.predict(X_test)

# Calculate accuracy
accuracy = accuracy_score(y_test, y_pred)
print(f'Accuracy: {accuracy}')

Step 6: Visualize Feature Importance

Understand which features contributed most to the model's decisions by plotting feature importance.


# Plot feature importance
importances = model.feature_importances_
plt.figure(figsize=(10,6))
plt.barh(X.columns, importances)
plt.title('Feature Importance')
plt.show()
