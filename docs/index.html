Welcome to Python Programming

In this tutorial, we will cover the basics of Python programming. Whether you're new to programming or just looking to refresh your skills, this guide will help you get started with Python.

Why Learn Python?

Python is an easy-to-learn, powerful programming language that you can use to build a wide range of applications. From web development to data science, Python is one of the most versatile languages out there.

Setting Up Python

To begin programming with Python, you need to install Python and set up a development environment.

Installation

sudo apt install python3

Alternatively, you can download the installer from the official Python website: https://www.python.org/downloads/

Your First Python Program

Let's start by writing a simple program. Create a new file called hello.py and add the following code:

print("Hello, World!")

When you run this program, it will print Hello, World! to the console. It's the most basic program in Python!

Variables and Data Types

In Python, you can store values in variables. Here are some basic data types:


age = 25 # Integer
height = 1.75 # Float
name = "Jessica" # String
is_learning = True # Boolean

print("Name:", name)
print("Age:", age)
print("Height:", height)
print("Learning Python:", is_learning)

Keep experimenting and building more Python projects!
